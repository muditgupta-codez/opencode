# Placeholder Icons

These are placeholder icons from opencode. Replace them with Forge Studio branded icons.

To generate new icons:

1. Save source image as `app-icon.png` in `packages/desktop`
2. Generate PNGs at all required sizes (32, 64, 128, 256, 512)
3. Create `icon.ico` containing multiple sizes (16, 32, 48, 256)
4. Create `icon.icns` using Image2Icon or similar tool
